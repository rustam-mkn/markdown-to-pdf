# Markdown to PDF Cards

Инструмент для генерации профессиональных PDF-документов с карточным дизайном из HTML-контента.

## Описание

Проект конвертирует HTML-файлы с карточками в векторные PDF-документы. Каждая карточка становится отдельной страницей в итоговом PDF-файле с автоматической подстройкой высоты под контент.

## Особенности

- Векторный PDF с возможностью выделения и копирования текста
- Профессиональный дизайн карточек с градиентами и стилизацией
- Автоматическое масштабирование карточек по содержимому
- Поддержка титульных и обычных карточек
- Стилизованные таблицы и info-блоки

## Технологии

- **Node.js** - среда выполнения
- **Puppeteer** - конвертация HTML в векторный PDF
- **pdf-lib** - объединение страниц PDF

## Установка

```bash
npm install
```

## Использование

### Генерация одного PDF-файла

```bash
node scripts/create-pdf.js
```

### Генерация всех PDF-файлов

```bash
node scripts/create-all-pdfs.js
```

Результат будет сохранен в папке `dist/`.

## Структура проекта

```
source/
  html/         - HTML-файлы с карточками
scripts/
  create-pdf.js - Скрипт генерации одного PDF
  create-all-pdfs.js - Скрипт генерации всех PDF
dist/           - Готовые PDF-файлы
```
